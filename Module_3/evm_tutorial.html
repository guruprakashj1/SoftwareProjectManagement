üéØ What is Earned Value Management?

üéØ Purpose

EVM integrates scope, schedule, and cost to measure project performance and predict future outcomes. It answers: "How much work should be done, how much is actually done, and what did it cost?"

üìö Key EVM Metrics

üìÖ Planned Value (PV)

What it is: The budgeted cost of work scheduled to be completed by a specific date.

Example: By week 6, we planned to complete $76,000 worth of work.

PV = Planned % Complete √ó Total Budget

‚úÖ Earned Value (EV)

What it is: The budgeted cost of work actually completed.

Example: We actually completed $74,000 worth of work by week 6.

EV = Actual % Complete √ó Total Budget

üí∞ Actual Cost (AC)

What it is: The total cost incurred for the work completed.

Example: We spent $69,000 to complete the work.

AC = All costs incurred to date

üìä Schedule Performance Index (SPI)

What it is: Measures schedule efficiency.

Interpretation:
‚Ä¢ SPI > 1.0 = Ahead of schedule
‚Ä¢ SPI = 1.0 = On schedule
‚Ä¢ SPI < 1.0 = Behind schedule

SPI = EV √∑ PV

üíµ Cost Performance Index (CPI)

What it is: Measures cost efficiency.

Interpretation:
‚Ä¢ CPI > 1.0 = Under budget
‚Ä¢ CPI = 1.0 = On budget
‚Ä¢ CPI < 1.0 = Over budget

CPI = EV √∑ AC

üéØ Estimate at Completion (EAC)

What it is: Predicted final project cost based on current performance.

Use: Helps forecast if project will exceed budget.

EAC = Total Budget √∑ CPI

üßÆ Step-by-Step Calculations

Step 1: Calculate Schedule Performance Index (SPI)

Given:

  • Planned Value (PV) = $76,000
  • Earned Value (EV) = $74,000
SPI = EV √∑ PV = $74,000 √∑ $76,000 = 0.97

Schedule Variance: (1.0 - 0.97) √ó 100 = 3% behind schedule

Interpretation: We achieved 97% of planned schedule performance, so we're 3% behind

Step 2: Calculate Cost Performance Index (CPI)

Given:

  • Earned Value (EV) = $74,000
  • Actual Cost (AC) = $69,000
CPI = EV √∑ AC = $74,000 √∑ $69,000 = 1.07

Cost Efficiency: (1.07 - 1.0) √ó 100 = 7% under budget

Interpretation: We get $1.07 worth of work for every $1.00 spent, so 7% cost efficiency gain

Step 3: Calculate Estimate at Completion (EAC)

Given:

  • Total Budget = $100,000
  • Cost Performance Index (CPI) = 1.07
EAC = Total Budget √∑ CPI = $100,000 √∑ 1.07 = $93,458

Interpretation: Based on current performance, the project will cost $93,458 instead of $100,000

üßÆ Understanding Percentage Calculations

üìÖ Schedule Variance Formula

Schedule Variance % = (1.0 - SPI) √ó 100

Example: (1.0 - 0.97) √ó 100 = 3%

Logic: Perfect performance = 1.0, so any difference shows variance

üí∞ Cost Efficiency Formula

Cost Efficiency % = (CPI - 1.0) √ó 100

Example: (1.07 - 1.0) √ó 100 = 7%

Logic: Values above 1.0 show positive efficiency

üéØ Quick Reference:

  • SPI = 1.15: (1.15 - 1.0) √ó 100 = 15% ahead of schedule

  • CPI = 1.20: (1.20 - 1.0) √ó 100 = 20% under budget
