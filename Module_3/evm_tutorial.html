📊 Earned Value Management (EVM)

Interactive Tutorial - Master Project Control Through Real-World Examples

🎯 What is Earned Value Management?

🎯 Purpose

EVM integrates scope, schedule, and cost to measure project performance and predict future outcomes. It answers: "How much work should be done, how much is actually done, and what did it cost?"

📈 Benefits

• Early warning system for project issues
• Objective performance measurement
• Reliable forecasting
• Better decision-making support

🔧 Applications

Used in construction, software development, manufacturing, aerospace, and any project where you need to track progress against budget and schedule.

🏗️ Tutorial Project: Office Building Construction

Project Details:

  • Total Budget: $100,000
  • Duration: 10 weeks
  • Current Status: Week 6 completed
  • Planned Work Value: $76,000 (76% of project)
  • Actual Work Completed: $74,000 worth
  • Money Spent: $69,000

We'll use this project throughout the tutorial to demonstrate EVM concepts.

📚 Key EVM Metrics

📅 Planned Value (PV)

What it is: The budgeted cost of work scheduled to be completed by a specific date.

Example: By week 6, we planned to complete $76,000 worth of work.

PV = Planned % Complete × Total Budget

✅ Earned Value (EV)

What it is: The budgeted cost of work actually completed.

Example: We actually completed $74,000 worth of work by week 6.

EV = Actual % Complete × Total Budget

💰 Actual Cost (AC)

What it is: The total cost incurred for the work completed.

Example: We spent $69,000 to complete the work.

AC = All costs incurred to date

📊 Schedule Performance Index (SPI)

What it is: Measures schedule efficiency.

Interpretation:
• SPI > 1.0 = Ahead of schedule
• SPI = 1.0 = On schedule
• SPI < 1.0 = Behind schedule

SPI = EV ÷ PV

💵 Cost Performance Index (CPI)

What it is: Measures cost efficiency.

Interpretation:
• CPI > 1.0 = Under budget
• CPI = 1.0 = On budget
• CPI < 1.0 = Over budget

CPI = EV ÷ AC

🎯 Estimate at Completion (EAC)

What it is: Predicted final project cost based on current performance.

Use: Helps forecast if project will exceed budget.

EAC = Total Budget ÷ CPI

🧮 Step-by-Step Calculations

Step 1: Calculate Schedule Performance Index (SPI)

Given:

  • Planned Value (PV) = $76,000
  • Earned Value (EV) = $74,000
SPI = EV ÷ PV = $74,000 ÷ $76,000 = 0.97

Schedule Variance: (1.0 - 0.97) × 100 = 3% behind schedule

Interpretation: We achieved 97% of planned schedule performance, so we're 3% behind

Step 2: Calculate Cost Performance Index (CPI)

Given:

  • Earned Value (EV) = $74,000
  • Actual Cost (AC) = $69,000
CPI = EV ÷ AC = $74,000 ÷ $69,000 = 1.07

Cost Efficiency: (1.07 - 1.0) × 100 = 7% under budget

Interpretation: We get $1.07 worth of work for every $1.00 spent, so 7% cost efficiency gain

Step 3: Calculate Estimate at Completion (EAC)

Given:

  • Total Budget = $100,000
  • Cost Performance Index (CPI) = 1.07
EAC = Total Budget ÷ CPI = $100,000 ÷ 1.07 = $93,458

Interpretation: Based on current performance, the project will cost $93,458 instead of $100,000

🧮 Understanding Percentage Calculations

📅 Schedule Variance Formula

Schedule Variance % = (1.0 - SPI) × 100

Example: (1.0 - 0.97) × 100 = 3%

Logic: Perfect performance = 1.0, so any difference shows variance

💰 Cost Efficiency Formula

Cost Efficiency % = (CPI - 1.0) × 100

Example: (1.07 - 1.0) × 100 = 7%

Logic: Values above 1.0 show positive efficiency

🎯 Quick Reference:

  • SPI = 0.90: (1.0 - 0.90) × 100 = 10% behind schedule
  • SPI = 1.15: (1.15 - 1.0) × 100 = 15% ahead of schedule
  • CPI = 0.85: (1.0 - 0.85) × 100 = 15% over budget
  • CPI = 1.20: (1.20 - 1.0) × 100 = 20% under budget

🏗️ Complete Project Example

📋 Project: Mobile App Development

Scenario: A software company is developing a mobile banking app.

📊 Project Details

  • Total Budget: $500,000
  • Duration: 6 months
  • Current: Month 4 (67% timeline)
  • Team: 12 developers

📈 Current Status

  • Planned Work: $335,000 (67%)
  • Completed Work: $310,000 (62%)
  • Money Spent: $340,000
  • Features: 155 of 250 complete

Step-by-Step Analysis

1. Calculate Schedule Performance

SPI = EV ÷ PV = $310,000 ÷ $335,000 = 0.93

✅ Result: Project is 7% behind schedule

2. Calculate Cost Performance

CPI = EV ÷ AC = $310,000 ÷ $340,000 = 0.91

⚠️ Result: Project is 9% over budget

3. Forecast Final Cost

EAC = Budget ÷ CPI = $500,000 ÷ 0.91 = $549,451

🔮 Result: Project will likely cost $49,451 more than planned

🎯 Project Manager Decisions

🚨 Immediate Actions

  • Review resource allocation
  • Identify bottlenecks
  • Consider overtime or additional resources
  • Reassess scope priorities

📊 Stakeholder Communication

  • Report 7% schedule delay
  • Inform about 9% cost overrun
  • Present recovery options
  • Request additional budget approval

🔄 Recovery Strategies

  • Fast-track critical features
  • Reduce scope if possible
  • Improve team efficiency
  • Negotiate timeline extension
