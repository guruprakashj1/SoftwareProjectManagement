IT Project Management

Module 2: Project Scheduling and Cost Control

Course Code: CSE4033 | Duration: 6 Hours

Module Overview

This module focuses on advanced project scheduling techniques and cost control mechanisms essential for effective IT project management. You will learn about critical path methods, resource allocation strategies, and techniques to optimize project timelines and budgets.

Learning Objectives

  • Understand precedence relationships in project scheduling
  • Differentiate between PERT and CPM methodologies
  • Calculate float and understand its significance in project management
  • Apply crashing techniques to reduce project duration
  • Implement project cost control using PERT/Cost
  • Manage resources effectively through scheduling and leveling

Key Topics

1. Project Scheduling 2 Hours

Precedence Relationship: Understanding task dependencies and how they affect project timelines.

Difference between PERT and CPM:

  • PERT (Program Evaluation and Review Technique) - probabilistic approach
  • CPM (Critical Path Method) - deterministic approach
  • When to use each methodology

Float Calculation:

  • Types of float (total float, free float, project float)
  • How float impacts schedule flexibility
  • Using float for resource optimization

Cost reduction by Crashing:

  • Identifying crashable activities
  • Calculating crash cost vs. time trade-offs
  • Optimizing project duration within budget constraints

2. Cost Control and Scheduling 4 Hours

Project Cost Control (PERT/Cost):

  • Integrating cost analysis with schedule management
  • Earned value management concepts
  • Cost performance measurement

Resource Scheduling:

  • Allocating resources to project activities
  • Resource-constrained vs. time-constrained scheduling
  • Resource histograms and allocation charts

Resource Leveling:

  • Smoothing resource demand over time
  • Techniques for avoiding resource overload
  • Impact of leveling on project duration

Practical Applications

This module includes hands-on exercises where you will:

  • Create network diagrams with precedence relationships
  • Calculate critical paths and floats for sample projects
  • Perform crashing analysis to optimize project schedules
  • Develop resource-loaded schedules and perform leveling

Preparation Materials

Before attending this module, please review:

  • Basic project scheduling concepts from Module 1
  • Network diagram fundamentals
  • Project cost estimation techniques
